# 💰 Expense Tracker - Gestor de Gastos Inteligente

Una aplicación moderna para registrar, clasificar y analizar tus gastos diarios de forma visual e intuitiva.

## 🎯 Características

- **📝 Registro de Gastos**: Ingresa tus gastos de forma rápida y sencilla
- **🏷️ Clasificación Automática**: Los gastos se clasifican automáticamente por categoría
- **📊 Dashboard Visual**: Gráficos interactivos y resumen de gastos
- **📈 Estadísticas**: Análisis detallado de tus patrones de gasto
- **💾 Almacenamiento Local**: Tus datos se guardan en tu navegador
- **📱 Responsive Design**: Funciona perfectamente en dispositivos móviles

## 🛠️ Tecnologías Utilizadas

- **Frontend**: React + TypeScript
- **Styling**: Tailwind CSS
- **Gráficos**: Chart.js
- **Almacenamiento**: LocalStorage
- **Build Tool**: Vite

## 📦 Instalación

```bash
npm install
npm run dev
```

## 🚀 Uso

1. **Agregar Gasto**: Haz clic en "Nuevo Gasto"
2. **Ingresa Detalles**: Cantidad, descripción, categoría y fecha
3. **Visualiza Resumen**: Dashboard con gráficos y estadísticas
